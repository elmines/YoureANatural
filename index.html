Module One: Navigating Sheet Music

For this lesson, we'll go over sheet music notation, so you're up to speed when we start working with the staff in later lessons.


Staff: 5 lines and 4 spaces on which most of the notes are located. As we will learn later a note’s location on the staff indicates the pitch to be played

Clef This is a treble clef. The clef sign tells the musician how to interpret the note’s location on the staff.

Measure A measure is a time segment made up of a specific number of beats defined at the beginning of a piece of music. Measures are separated by vertical bars in the music End bar This signals the end of a piece of music

Time Signature This specifies the properties of a measure of music. The top number specifies the number of beats in a measure. The bottom number specifies which note is given the full value of a beat. For example in 3/4 time there are 3 beats per measure and the quarter not (¼) is takes up a full beat.

Key Signature This tell the musician which “value” of the note should be assumed throughout the entire piece. You will learn more about key signatures in module 4.
